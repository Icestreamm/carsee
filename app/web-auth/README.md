# CarSee — browser password reset

Static page for Supabase password recovery emails (opens in Chrome/Safari, not the app).

## Render Static Site

| Setting | Value |
|---------|--------|
| Repository | `lcestreamm/carsee` |
| Root Directory | `app/web-auth` |
| Publish Directory | `.` |
| Build Command | *(empty)* |

Live URL: `https://carsee-auth.onrender.com/reset-password.html`

Add that URL (+ trailing-slash variant) in Supabase → Authentication → URL Configuration → Redirect URLs.
